## 特徴

<CardGrid stagger>
  <Card title="シンプルなAPI" icon="pencil">
    PDFページを画像に変換するための数個の関数のみ。複雑な設定は不要です。
  </Card>
  <Card title="メモリ効率" icon="rocket">
    非同期ジェネレータを使用してページを1つずつ処理。大きなPDFもメモリ問題なく処理できます。
  </Card>
  <Card title="CJKサポート" icon="document">
    中国語、日本語、韓国語フォントを標準でサポート。
  </Card>
  <Card title="TypeScript" icon="setting">
    TypeScriptで記述され、完全な型定義を提供。IDEサポートも万全です。
  </Card>
</CardGrid>
